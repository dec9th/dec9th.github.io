[On Debian/Ubuntu]

```
apt-get update
apt-get install -y net-tools procps iputils-ping netcat
```

- `net-tools` :` ifconfig`, `netstat`를 포함한 commands
- `procps` : `ps`를 포함한 commands
- `iputils-ping` : `ping`를 포함한 commands
- `netcat` : `nc` commands
